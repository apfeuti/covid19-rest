COVID19 REST-API

This is a REST-API to get data about COVID19 (new corona-virus) cases.

The data can be filtered by country, area, (city/admin on JHU-data) and date. Aggregated sums are available on each filter level. Output is available in json and csv.

API-Documentation

Please refer to https://github.com/apfeuti/covid19-rest

Data sources

    Open Data Kanton Zürich (openZH) (for detailed figures about Switzerland)
    Johns Hopkins University (for figures about the world - including detailed data on city-level for the United States of America (USA)).
